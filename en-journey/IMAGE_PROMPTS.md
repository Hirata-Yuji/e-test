## Usage Notes

- For **Midjourney**: copy the prompt text directly into `/imagine`
- For **DALL-E 3**: remove the `--ar` and `--v 6` suffix flags
- For **Adobe Firefly**: use the descriptive text; specify aspect ratio in UI
- Recommended image size: export at 2× the display size for Retina screens
