![Growcal](https://img.shields.io/badge/Status-Active-success)
![Version](https://img.shields.io/badge/Version-1.0.0-blue)

## 📖 プロジェクト概要

**Growcal**（グロウカル）は、「Growth（成長）」と「LOCAL（ローカル）」を組み合わせた造語です。

グローバルな視点でローカルビジネスを行っている人や企業、モノコトを紹介するメディアサイトとして、地域の成長を加速することをミッションとしています。

## ✨ 主な機能

### 実装済み機能

1. **レスポンシブデザイン**
   - デスクトップ、タブレット、モバイルに完全対応
   - モバイルハンバーガーメニュー実装

2. **ヒーローセクション**
   - 地球の画像を背景にした印象的なヒーローエリア
   - オーバーレイ効果で視認性向上

3. **PICK UP COMPANYセクション**
   - 6つの企業ロゴ表示（グリッドレイアウト）
   - ホバーエフェクト
   - Official Media Partner表記

4. **NEWSセクション**
   - 日付付きニュース一覧（YYYY-MM-DD形式）
   - 最新4件のニュース表示
   - クリーンなリストデザイン

5. **記事一覧セクション**
   - 3カラムグリッドレイアウト
   - 6つの記事カード（仮コンテンツ）
   - カテゴリーラベル表示

6. **カテゴリー分類**
   - BUSINESS（ビジネス）
   - TECHNOLOGY（テクノロジー）
   - LIFESTYLE（ライフスタイル）
   - INTERVIEW（インタビュー）

7. **インタラクティブ機能**
   - スムーススクロール
   - ホバーアニメーション効果
   - フェードインアニメーション
   - 画像遅延読み込み最適化
   - ナビゲーションメニュー（モバイル対応）

## 🎨 デザイン

### カラースキーム
- **プライマリ**: `#1a1a1a`（ダークグレー）
- **アクセント**: `#007bff`（ブルー）
- **背景**: `#ffffff`, `#f8f9fa`（ホワイト、ライトグレー）
- **テキスト**: `#333333`, `#666666`（グレー階調）

### フォント
- **日本語**: Noto Sans JP
- **英語**: Inter
- Google Fontsから読み込み

### レイアウト
- モダンなグリッドシステム
- カード型デザイン
- 視認性の高いタイポグラフィ
- 地球をモチーフにしたヒーローセクション

| デバイス | ブレークポイント | レイアウト |

| デスクトップ | 968px以上 | 6カラム企業グリッド、3カラム記事グリッド |
| タブレット | 768px - 968px | 3カラム企業グリッド、2カラム記事グリッド |
| モバイル | 768px以下 | 2カラム企業グリッド、1カラム記事グリッド |
| 小型モバイル | 480px以下 | 1カラム企業グリッド、最適化された1カラム |

## 🌟 Growcalのコンセプト

「Growth + LOCAL」= **Growcal**

グローバルな視点で地域のビジネスを見つめ、その成長を加速させる。  
世界と地域をつなぎ、持続可能な成長を実現するための情報プラットフォームです。
