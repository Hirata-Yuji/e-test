# 2026年春闘 新卒初任給リサーチレポート

**調査日**: 2026年3月25日
**対象**: 2026年4月入社の大卒新卒初任給（春闘動向を反映）

---

## 1. 調査概要

2026年春闘では、連合が賃上げ要求を「定昇込み5%以上」（中小は6%以上）と掲げ、賃上げ率は全体で約5.45%と3年連続の高水準が続いている。新卒初任給も歴史的な引き上げラッシュが起きており、大手企業を中心に「初任給30万円」が新常識となりつつある。

### 主要データソース
- マイナビ「2026年卒 企業新卒採用予定調査」
- 帝国データバンク「初任給に関する企業動向アンケート（2026年度）」
- HRog「26卒ナビサイト初任給調査」
- 厚生労働省「令和6年賃金構造基本統計調査」
- 労務行政研究所「東証プライム上場企業197社 初任給調査（2025年度速報）」
- 第一生命経済研究所「2026年春闘賃上げ率見通し」
- 東京商工リサーチ「2026年度 賃上げ実施予定調査」

## 2. グループ別 新卒初任給データ（大卒）

### 2-1. 上場企業

#### 全上場企業（マイナビ調査 2026年卒）

| 指標 | 金額 |
|------|------|

| **中央値** | **約23.0〜24.0万円**（推定） |

- 上場企業の88.8%が初任給引き上げを実施。
- 引き上げ額で最も多いのは「5,000円〜1万円未満」（40.3%）。

| **平均（院卒修士）** | **273,327円** |

- 「全学歴引き上げ」企業が83.2%と2年連続で8割超。
- 2026年度はさらに引き上げが進んでおり、平均は**26万円台前半〜半ば**に到達していると推定。
- 30万円以上を提示するプライム企業が急増（三菱UFJ銀行、三井住友銀行、塩野義製薬、セガ、ニトリHD等）。

- スタンダード市場に特化した公的調査は現時点で確認されていない。
- 中堅上場企業の位置づけであり、帝国データバンクの「大企業」カテゴリに一部含まれる。

#### 東証グロース市場（推定）

| 指標 | 金額 |
|------|------|
| **平均** | **約23.0〜28.0万円**（企業間のばらつきが大きい） |
| **中央値** | **約23.0〜25.0万円**（推定） |

- グロース市場は新興・ベンチャー企業が中心で、企業によって初任給水準の差が大きい。
- IT・SaaS系は高めに設定する傾向（25〜30万円以上）がある一方、創業期の企業は20万円台前半の場合もある。

> **注意**: 東証の市場区分（プライム・スタンダード・グロース）別に初任給を比較した公的統計データは2026年3月時点で存在しない。上記のスタンダード・グロースのデータは、帝国データバンク・マイナビ等の企業規模別データ、及び個別企業の公開情報から推定したものである。

| 指標 | 金額 |
|------|------|

| **中央値** | **約21.5〜22.0万円**（推定） |

#### 東京商工リサーチ調査

| 指標 | データ |
|------|--------|
| **賃上げ実施予定企業** | **82.8%**（大企業93.8%との差 11.0pt） |
| **賃上げ率5%以上** | 全体35.5%だが中小単独では5%未達が多い |
| **賃上げ率6%以上（連合目標）** | **わずか7.2%** |

#### 主な課題
- 従業員300人未満の企業は賃上げ率が2年連続で5%に届かず、大企業との格差が拡大中。
- 「新卒者の初任給増額」実施率は中小企業21.6%に対し大企業39.5%（17.9ptの差）。
- 原資確保が困難な小規模企業では引き上げが半数にとどまる。

### 2-3. スタートアップ（ベンチャー企業）

#### 一般的な傾向

| 指標 | 金額 |
|------|------|
| **平均** | **約25万〜28万円** |
| **中央値** | **約25万円前後**（推定） |
| **平均年収** | **約382万円** |

- 一般企業の初任給（約21万円）より約4万円高い水準。
- 少数精鋭型のため、優秀人材確保のために高めに設定する傾向。
- ただし福利厚生（社宅補助、家賃補助等）は大手より手薄なケースが多く、実質的な待遇は額面ほどの差がない場合もある。

#### 初任給30万円以上のスタートアップ・ベンチャーの動向
- IT・SaaS・コンサル系を中心に30万円以上を提示する企業が増加。
- GMOインターネットグループは「新卒年収710万プログラム」で月額591,675円（条件付き）。
- Rapidusは433,400円を提示。

#### スタートアップ特有の報酬体系
- ストックオプション（SO）を含む報酬パッケージが一般的であり、月額初任給だけでは比較しにくい。
- 成果連動型の給与体系を採用する企業が多く、入社後の昇給ペースが大企業より速い場合がある。

> **注意**: 「スタートアップ」に特化した大卒初任給の統計的調査データは2026年3月時点で確認されていない。上記はベンチャー企業全般のデータ及び個別企業の公開情報に基づく。

## 5. 2026年春闘の背景と今後の見通し

### 賃上げ率の推移
- 2024年春闘: 5.58%
- 2025年春闘: 5.39%
- 2026年春闘（見通し）: 約5.45%（第一生命経済研究所予測）

### 主な背景要因
1. **人材獲得競争の激化**: 少子化による構造的な人手不足が深刻化
2. **物価上昇への対応**: 実質賃金のプラス定着に向けた動き
3. **政府の賃上げ要請**: 経済好循環の実現に向けた後押し
4. **企業間競争**: 他社の引き上げに対する追随圧力（「他企業が引き上げをしているため」51.2%）

### 今後の課題
- **中小企業の格差是正**: 大企業との賃上げ率・水準格差が拡大傾向
- **賃金カーブのフラット化**: 初任給急騰により中堅社員との逆転・接近現象が発生
- **持続可能性**: 賃上げ促進税制の大企業向け廃止（2026年3月末）の影響
- **価格転嫁**: 中小企業の賃上げ原資確保の困難さ

## 6. データの制約事項

1. **市場区分別データの不在**: 東証プライム・スタンダード・グロース市場区分別の初任給統計は公的機関から発表されていない。スタンダード・グロースの数値は推定値である。
2. **スタートアップの定義**: 「スタートアップ」の明確な定義が統計ごとに異なり、ベンチャー企業全般のデータと混在している。
3. **中央値の不在**: 多くの調査が平均値のみを公表しており、中央値は分布データから推定したものが含まれる。
4. **調査時点のずれ**: 各調査の実施時期が異なるため、数値に若干の差がある。
5. **春闘進行中**: 2026年3月時点では春闘交渉が継続中であり、最終的な妥結額は変動する可能性がある。
