# LEADERS VISION - 経営者メディアサイト

## プロジェクト概要

経営者やリーダーのストーリーを切り抜くインタビューメディアサイト。
ダークテーマと近未来的デザインで、ミニマルかつ映画的な雰囲気を演出します。

## デザインコンセプト

- **カラースキーム**: ダークテーマ（黒・グレーベース）+ 赤のアクセントカラー
- **雰囲気**: ミニマル、映画的、近未来的
- **特徴**: 幾何学的パターン、グリッドレイアウト、視覚的インパクト
- **レスポンシブ**: モバイル・タブレット・デスクトップ対応

## カスタマイズ方法

### カラーの変更
`css/style.css`の`:root`セクションでカラー変数を編集：
```css
:root {
    --primary-bg: #0a0a0a;
    --secondary-bg: #1a1a1a;
    --accent-red: #FF0033;
    --text-primary: #ffffff;
    --text-secondary: #b0b0b0;
}
```

### コンテンツの追加
- 新しい記事: `article*.html`をコピーして編集
- メニュー項目: 各HTMLファイルの`<nav>`セクションを更新
