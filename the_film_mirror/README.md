# FILMS - 経営者メディアサイト

経営者やリーダーのストーリーを切り抜くインタビューメディアサイト。

## プロジェクト概要

「FILMS」は、THE FILMを参考にした、経営者やリーダーのストーリーにフォーカスしたメディアサイトです。
ダークテーマと近未来的なデザインで、映画的な雰囲気を持つミニマルなレイアウトを実現しています。

## 完成機能

### ページ構成
- **トップページ（index.html）** - ヒーローセクション、コンセプト、6記事のマガジンレイアウト
- **記事詳細ページ（article1-6.html）** - 2カラムレイアウト（画像+企業情報 | 記事本文）
- **Aboutページ（about.html）** - メディアのビジョンと理念
- **Privacyページ（privacy.html）** - プライバシーポリシー
- **Infoページ（info.html）** - お問い合わせと運営情報

### デザイン特徴
- **ダークテーマ**: 黒・グレーベースに赤（#FF0033）のアクセントカラー
- **近未来的要素**: 幾何学的グリッドパターン、モダンなアニメーション
- **映画的な雰囲気**: ミニマルで洗練されたレイアウト、モノクロ写真
- **レスポンシブデザイン**: デスクトップ、タブレット、モバイル対応

### インタラクティブ機能
- ハンバーガーメニュー（モバイル）
- スムーズスクロール
- フェードインアニメーション
- SNSシェアボタン
- ホバーエフェクト

## カスタマイズ方法

### 記事の追加
1. 既存の記事ファイル（article1.html等）をコピー
2. 企業情報、人物名、記事内容を変更
3. index.htmlに新しい記事カードを追加

### 色の変更
`css/style.css`の`:root`セクションでカラー変数を変更:
```css
:root {
    --primary-bg: #0a0a0a;
    --accent-red: #FF0033;
    /* ... */
}
```

### フォントの変更
`index.html`のGoogle Fontsリンクを変更し、`style.css`の`--font-primary`を更新
