# FILMS - 経営者メディアサイト

## プロジェクト概要

経営者やリーダーのストーリーを切り抜くインタビューメディアサイト。
ダークテーマと近未来的デザインで、ミニマルかつ映画的な雰囲気を演出します。

## 最新アップデート（v2.1）

### 🎬 記事詳細ページの大幅リニューアル
- ✨ **2カラムレイアウト**：左に画像+企業情報、右に記事本文の画期的なデザイン
- 🖼️ **大型人物画像**：左カラム上部に印象的なモノクロ写真を配置
- 📋 **企業情報セクション**：役職、氏名、会社名、URLをシンプルに表示
- 📤 **統合シェアボタン**：左カラム下部にSNSシェアアイコンを配置
- 🎯 **スティッキーレイアウト**：左カラムを固定し、右カラムがスクロール

### ブランディング更新（v2.1）
- **サイト名**：「THE|FILM」から「FILMS」にシンプル化

### 以前のアップデート（v2.0）
- 🖼️ **フルスクリーンヒーロー**：初期画面に大きなロゴとスクロールインジケーター
- 📸 **コンセプトセクション**：背景画像とテキストオーバーレイの映画的演出
- 📰 **マガジンレイアウト**：長方形ボックスを組み合わせたモダンなグリッドデザイン
- 📋 **6記事に拡張**：サンプル記事を3つから6つに増加
- 🎨 **動的レイアウト**：小カード + 大カードの組み合わせでランダム感を演出

## デザインコンセプト

- **カラースキーム**: ダークテーマ（黒・グレーベース）+ 赤のアクセントカラー
- **雰囲気**: ミニマル、映画的、近未来的
- **特徴**: 幾何学的パターン、グリッドレイアウト、視覚的インパクト
- **レスポンシブ**: モバイル・タブレット・デスクトップ対応

## カスタマイズ方法

### カラーの変更
`css/style.css`の`:root`セクションでカラー変数を編集：
```css
:root {
    --primary-bg: #0a0a0a;
    --secondary-bg: #1a1a1a;
    --accent-red: #FF0033;
    --text-primary: #ffffff;
    --text-secondary: #b0b0b0;
}
```

### コンテンツの追加
- 新しい記事: `article*.html`をコピーして編集
- メニュー項目: 各HTMLファイルの`<nav>`セクションを更新
